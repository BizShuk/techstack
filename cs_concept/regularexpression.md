# Regular Expression

`[^/]`, anything not `/`
`.*?` or `.+?`, shortest match
`(?:abc)`, abc is optional
`.[^.]*`, `.*` 但不含 `..*`

### symbol

- ^, starting point
- $, ending point
- ., any char
- +, match previous char 1 or more time
- ?, match previous char 0 or 1 time
- *, match previous char 0 or more time
- ?:, optional
- \b, 字元邊界, 空白,標點,string開始/結尾
